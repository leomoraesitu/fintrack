# Inventario de Assets de Design

## Objetivo

Inventariar os arquivos recebidos do prototipo para manter rastreabilidade de branding e referencias visuais.

## Branding

### Logo

- arquivo PNG: `docs/design/assets/branding/logo/icone-fintrack.png`
- arquivo fonte: `docs/design/assets/branding/logo/icone-fintrack.psd`
- uso: icone principal da marca e referencia para app icon e avatar de marca

### Wordmark

## Regra de uso

- estes arquivos sao artefatos de documentacao e handoff visual
- eles nao substituem assets finais de runtime do app
- se um asset passar a ser usado pela aplicacao, criar copia apropriada na estrutura de runtime do Flutter
