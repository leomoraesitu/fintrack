# Backend e sincronizacao

## Status

Documento de evolucao futura.

## Objetivo

Registrar a direcao prevista caso o projeto avance para autenticacao real, persistencia remota e sincronizacao entre dispositivos.

## Possiveis motivadores

- necessidade de multiusuario
- sincronizacao entre dispositivos
- backup remoto dos dados
- autenticacao real

## Pre-condicoes para iniciar

- MVP local concluido e estavel
- dominio e persistencia local consolidados
- regras de sincronizacao claramente definidas

## Perguntas que precisam ser respondidas antes da implementacao

- qual backend sera usado
- como sera feita a autenticacao real
- qual sera a estrategia de sincronizacao e resolucao de conflito
- quais dados sao sincronizados e com que frequencia

## Limite atual

Nenhum backend ou sincronizacao faz parte do escopo atual do projeto.
